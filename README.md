# Prem's Claude Code Quickstart

A single-file static site of [Prem's opinionated Claude Code guide](https://puiwah.github.io/claude-code-quickstart/), built from `content.md` for a meditation-community audience.

## Editing

1. Edit `content.md` (the source of truth — every word of prose lives there).
2. Run `node build.js` to regenerate `index.html` from the markdown.
3. Commit and push — GitHub Pages auto-deploys from `main`.

`BUILD_SPEC.md` is the handoff spec used to generate this site, kept around so future contributors (or future Claude Code sessions) can rebuild from scratch.
